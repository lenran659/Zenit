# Zenit 项目管理系统

## 🎨 设计理念

极简主义的项目管理工具，灵感来自 Linear，采用暗黑模式设计：

- **深 Zinc-900 背景** - 专业的暗黑主题
- **Slate-800 边框** - 精致的分隔线
- **电光蓝 (Cyan-500)** - 关键操作按钮
- **毛玻璃效果** - 现代化的视觉层次
- **4px 圆角** - 统一的设计语言

## 📁 项目结构

```
app/
├── components/          # UI 组件
│   ├── Sidebar.tsx     # 可折叠侧边栏（毛玻璃效果）
│   ├── HomePage.tsx    # 首页 - 工作概览
│   ├── Dashboard.tsx   # 任务页 - 看板视图
│   ├── AnalyticsPage.tsx # 分析页 - 数据统计
│   ├── SettingsPage.tsx  # 设置页 - 偏好配置
│   ├── KanbanBoard.tsx # 看板视图（拖拽功能）
│   ├── TaskCard.tsx    # 任务卡片
│   └── CommandPalette.tsx # Cmd+K 命令面板
├── hooks/
│   └── useProjectStore.ts # 项目状态管理和业务逻辑
├── types.ts            # TypeScript 类型定义
├── page.tsx            # 主页面（路由控制）
├── layout.tsx          # 布局配置
└── globals.css         # 全局样式
```

## 🚀 核心功能

### 1. 首页 (HomePage)
**业务逻辑：**
- 显示工作概览和统计数据
- 我的任务列表（按分配人筛选）
- 今日到期任务（按日期筛选）
- 紧急任务列表（按优先级筛选）
- 项目完成率计算
- 团队成员统计

**功能：**
- 快速查看个人任务
- 实时统计卡片
- 项目进度可视化

### 2. 任务页 (Dashboard + KanbanBoard)
**业务逻辑：**
- 看板式任务管理
- 拖拽更新任务状态
- 按状态分组显示
- 任务统计（总数、进行中、已完成）
- 筛选和日期选择

**功能：**
- 5 列看板（待办、计划中、进行中、审核中、已完成）
- 拖拽任务卡片
- 新建任务按钮
- 任务详情展示

### 3. 分析页 (AnalyticsPage)
**业务逻辑：**
- 任务状态分布统计
- 优先级分布分析
- 团队成员任务分配统计
- 完成任务数量追踪
- 平均完成时间计算
- 本周完成任务统计

**功能：**
- 4 个关键指标卡片
- 状态分布进度条
- 优先级分布可视化
- 团队成员表现展示

### 4. 设置页 (SettingsPage)
**业务逻辑：**
- 用户信息管理
- 通知偏好设置
- 外观主题配置
- 隐私和安全设置
- 数据备份管理

**功能：**
- 个人信息编辑
- 通知开关（邮件、推送、提醒、周报）
- 主题和强调色选择
- 紧凑模式切换
- 数据导出和账户删除

### 业务逻辑核心 (useProjectStore)

1. **任务管理**
   - `addTask()` - 创建新任务
   - `updateTask()` - 更新任务信息
   - `updateTaskStatus()` - 更新任务状态
   - `deleteTask()` - 删除任务

2. **数据查询**
   - `getTasksByStatus()` - 按状态筛选任务
   - `sortTasksByPriority()` - 按优先级排序

3. **任务属性**
   - 优先级：urgent, high, medium, low
   - 状态：backlog, todo, in-progress, review, done
   - 分配人、截止日期、描述等

## 🎯 使用方法

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 页面导航

点击左侧边栏的图标切换不同页面：
- 首页 - 查看工作概览
- 任务 - 管理看板和任务
- 分析 - 查看数据统计
- 设置 - 配置个人偏好

### 拖拽任务

在任务页面的看板视图中，直接拖动任务卡片到不同的状态列即可更新任务状态。

### 命令面板

点击命令面板或使用 Cmd+K 快捷键打开搜索界面。

## 🎨 设计特点

- **无渐变** - 纯扁平化设计
- **高对比度** - 优秀的可读性
- **精致图标** - Lucide React 图标库
- **平滑过渡** - 所有交互都有流畅动画
- **响应式** - 适配不同屏幕尺寸
- **可折叠侧边栏** - 节省屏幕空间

## 🔧 技术栈

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React (图标)
